# 底部 Tab 栏 flex 垂直居中重构

- 操作时间：2026-04-21 09:53:56 +08:00
- 操作人：Codex
- 任务状态：已获批准，正在执行

## 1. 变更目标

- 按用户指定的结构与布局规则，重构微信小程序底部 `首页 + 我的` Tab 栏。
- 使用独立父容器承载两个 Tab 项，并通过 `display: flex`、`align-items: center`、`justify-content: space-around` 实现整体垂直居中。
- 每个 Tab 项内部使用列布局，确保图标与文字在列方向中稳定居中。
- 增加容错约束，避免文字长度变化导致图标位置偏移。

## 2. 问题分析

- 当前 `.liquid-tabbar` 同时承担背景容器和 Tab 布局容器职责，导致背景层样式与内容层居中规则耦合。
- 现有 `tab-item` 虽然已使用列布局，但缺少独立的内容容器与文字容错约束，文字变化时对视觉中心不够稳定。
- 需要通过结构拆分，把“背景胶囊”与“Tab 排布层”分离。

## 11. 容错处理说明

- 无论文案长度如何，Tab 的布局基准都由 `tab-content` 控制，而不是由单个文字节点临时撑开。
- 文案启用单行显示与截断策略，避免超长文字把图标从垂直中线挤开。
- 激活态仅保留轻微缩放，不保留向上位移，避免中线漂移。
- `tab-item` 不设置额外上下 padding，减少视觉偏差。
