# 我的页面顶部与底部导航栏磨砂效果调整日志

## 1. 目标

将小程序「我的」页面的顶部导航栏和底部自定义 tabBar 都调整为半透明磨砂风格。

- 顶部导航栏：原生系统导航栏不支持直接做真磨砂，因此改为 `navigationStyle: custom` 后自定义实现。
- 底部导航栏：项目里原本已经是自定义 tabBar，本次做了进一步的磨砂增强，让视觉风格与顶部保持一致。

## 2. 现状分析

### 2.1 顶部导航栏

- `pages/user/user.json` 原先只有 `navigationBarTitleText`，仍然使用系统导航栏。
- 系统导航栏无法直接实现 `backdrop-filter` 这类毛玻璃效果。
- 如果要做半透明磨砂效果，必须改成自定义导航栏。

### 2.2 底部导航栏

- 项目已经启用了 custom tabBar。
- 底部 tabBar 原本就有半透明背景和 blur 效果，但整体还可以再统一一点，让“磨砂感”更明显。

### 2.3 页面影响范围

- 这次只针对「我的」页面的顶部导航和全局底部 tabBar 视觉做调整。
- 其它业务页面暂不改动，避免扩散到不必要的范围。

## 4. 实际修改

### 4.1 顶部导航栏改为自定义

修改文件：

- `[wechat-app/pages/user/user.json](/G:/zhiximini/wechat-app/pages/user/user.json)`

修改内容：

- 新增 `navigationStyle: "custom"`。
- 保留原有 `navigationBarTitleText: "我的"` 作为页面标题语义信息。

这样做的原因：

- 原生导航栏无法实现真磨砂。
- 自定义导航栏可以直接叠加半透明背景、阴影和 `backdrop-filter`。

### 4.2 顶部导航栏布局与状态栏适配

修改文件：

- `[wechat-app/pages/user/user.js](/G:/zhiximini/wechat-app/pages/user/user.js)`
- `[wechat-app/pages/user/user.wxml](/G:/zhiximini/wechat-app/pages/user/user.wxml)`
- `[wechat-app/pages/user/user.wxss](/G:/zhiximini/wechat-app/pages/user/user.wxss)`

修改内容：

- 在页面 `data` 中新增：
  - `statusBarHeight`
  - `navContentHeight`
  - `navBarHeight`
- 在 `onLoad` 中初始化系统状态栏高度。
- 在页面顶部插入自定义导航栏视图。
- 导航栏使用半透明白底、边框、阴影和 `backdrop-filter` 来形成磨砂效果。
- 页面根容器增加顶部内边距，避免自定义导航栏遮挡正文内容。
- 导航栏本身设置为装饰层，不接收点击事件，避免影响页面滚动和下拉手势。

视觉特征：

- 导航栏保持居中标题「我的」。
- 采用轻微高光层，增强玻璃感。
- 标题颜色使用深色，保证在半透明背景上仍然清晰可读。

### 4.3 底部 tabBar 磨砂增强

修改文件：

- `[wechat-app/custom-tab-bar/index.wxss](/G:/zhiximini/wechat-app/custom-tab-bar/index.wxss)`

修改内容：

- 将 tabBar 阴影层的模糊半径从 `20rpx` 调整到 `24rpx`。
- 将 tabBar 主体背景从 `rgba(255, 255, 255, 0.5)` 调整为 `rgba(255, 255, 255, 0.42)`。
- 将 tabBar 边框从 `rgba(255, 255, 255, 0.3)` 调整为 `rgba(255, 255, 255, 0.34)`。
- 将 `backdrop-filter` / `-webkit-backdrop-filter` 从 `blur(20px) saturate(150%)` 调整为 `blur(24px) saturate(170%)`。

目的：

- 让底部 tabBar 的磨砂感更明显。
- 让顶部导航和底部导航在视觉上更统一。
