## 目标

- 提供管理员可登录的后台
- 支持商品审核、下架、用户禁用、反馈查看

## 必做任务

1. 初始化后台项目
2. 实现后台登录
3. 实现商品管理页
4. 实现用户管理页
5. 实现反馈管理页
6. 对接 CloudBase 数据

## 输出物

- 后台工程代码
- 商品审核能力
- 用户禁用能力
- 反馈查看能力

## 约束

- 不做复杂权限系统
- 不做客服工作台
- 不做统计大盘

## 验收标准

- 管理员能通过后台控制商品展示状态
- 管理员能禁用违规用户
