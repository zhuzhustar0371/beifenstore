# qa-devops-agent 任务包

## 你的职责

你负责闭环验证与预发布交付。

## 输入材料

- [00-command-board.md](g:\bishe2\docs\execution\00-command-board.md)
- [03-api-and-flow.md](g:\bishe2\docs\mvp\03-api-and-flow.md)

## 目标

- 验证发布、浏览、联系、回复、审核这条闭环
- 给出可交付的测试结论

## 必做任务

1. 列出冒烟测试清单
2. 验证登录流程
3. 验证发布流程
4. 验证审核通过后列表可见
5. 验证买家发起联系
6. 验证卖家回复
7. 验证后台下架与用户禁用
8. 输出预发布部署与回归建议

## 输出物

- 冒烟测试清单
- 测试执行结果
- 闭环验证报告
- 预发布说明

## 验收标准

- 能以用户身份和管理员身份分别走通主链路
- 明确记录阻塞问题和未实现项
