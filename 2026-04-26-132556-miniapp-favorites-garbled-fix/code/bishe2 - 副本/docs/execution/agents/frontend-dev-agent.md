# frontend-dev-agent 任务包

## 你的职责

你负责用户端 Uni-app MVP 的实际页面和交互落地。

## 输入材料

- [01-p0-pages.md](g:\bishe2\docs\mvp\01-p0-pages.md)
- [03-api-and-flow.md](g:\bishe2\docs\mvp\03-api-and-flow.md)
- [05-ui-reference-alignment.md](g:\bishe2\docs\mvp\05-ui-reference-alignment.md)

## 目标

- 初始化 Uni-app 项目
- 落地闲鱼风格的 MVP UI
- 优先做 `首页 -> 详情 -> 发布 -> 消息`

## 必做任务

1. 初始化 Uni-app 工程
2. 建立基础路由和页面目录
3. 实现首页商品流
4. 实现商品详情页
5. 实现发布页
6. 实现消息页和聊天页骨架
7. 接入 CloudBase：
   - 登录
   - 数据读取
   - 图片上传

## 输出物

- Uni-app 工程代码
- 基础 UI 组件
- CloudBase 接入层
- 可跑通的 P0 页面

## UI 约束

- 参考闲鱼黄黑白风格
- 保留搜索、卡片流、聊一聊、反馈入口
- 不做购买、地址、订单、担保交易

## 验收标准

- 用户能浏览商品
- 用户能发商品
- 用户能发起聊天
- UI 风格与参考图一致性明显，但功能范围仍是 MVP
