## 8. 常见问题排查
### 8.1 仍出现 "touristappid" 错误
- 检查微信开发者工具登录状态
- 检查项目根目录的 `project.config.json` 文件是否被正确识别
- 尝试清除开发者工具缓存并重启

### 8.2 无法连接服务器
- 检查服务器域名是否已添加到白名单
- 检查 HTTPS 证书是否有效
- 检查服务器防火墙设置

### 8.3 体验版无法访问
- 检查当前用户是否在体验版成员列表中
- 检查体验版版本是否已上传并设置

- [ ] 小程序能正常调用后端 API 接口
