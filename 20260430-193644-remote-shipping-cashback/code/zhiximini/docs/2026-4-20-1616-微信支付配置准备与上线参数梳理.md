## 本次识别出的微信支付必填参数

根据代码依赖，真正需要补齐的参数如下：

- `WECHAT_MINIAPP_APP_ID`
- `WECHAT_MINIAPP_APP_SECRET`
- `WECHAT_PAY_MCHID`
- `WECHAT_PAY_APPID`
- `WECHAT_PAY_API_V3_KEY`
- `WECHAT_PAY_SERIAL_NUMBER`
- `WECHAT_PAY_PRIVATE_KEY_PATH`
- `WECHAT_PAY_NOTIFY_URL`

其中说明：

- `WECHAT_MINIAPP_APP_ID`：小程序 AppID
- `WECHAT_MINIAPP_APP_SECRET`：小程序 AppSecret，用于 `jscode2session` 获取 `openid`
- `WECHAT_PAY_MCHID`：微信支付商户号
- `WECHAT_PAY_APPID`：支付使用的 AppID。当前实现预期与小程序 AppID 一致
- `WECHAT_PAY_API_V3_KEY`：微信支付 APIv3 密钥
- `WECHAT_PAY_SERIAL_NUMBER`：商户证书序列号
- `WECHAT_PAY_PRIVATE_KEY_PATH`：商户私钥文件路径，当前服务器建议使用 `/home/ubuntu/apps/backend-api/cert/apiclient_key.pem`
- `WECHAT_PAY_NOTIFY_URL`：支付回调地址，当前建议使用 `https://api.mashishi.com/api/pay/wechat/notify`
