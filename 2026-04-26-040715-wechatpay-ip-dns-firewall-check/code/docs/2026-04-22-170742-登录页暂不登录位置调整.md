# 2026-04-22 17:07 登录页“暂不登录”位置调整日志

## 1. 本次目标

调整 `知禧净享生活` 小程序登录页中“暂不登录”的显示位置与排版，让它稳定显示在“微信授权一键登录”按钮下方，并保持居中对齐，避免在真机上出现左偏或位置分散的问题。

## 2. 修改背景

当前登录页底部区域包含以下元素：

- “微信授权一键登录”主按钮
- “暂不登录”文本入口

从截图表现看，“暂不登录”虽然已经位于主按钮之后，但视觉上偏左、层级不够明确，容易让人误以为它和其他内容没有对齐到同一视觉中心。

- `wechat-app/pages/login/login.wxss`
