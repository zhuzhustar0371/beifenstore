# 管理端商品价格编辑功能

## 日期
2026-04-24

## 需求
管理端可以编辑商品价格，价格变动后多端同步更新，返现按比例同步调整。

## 现状分析

### 已有基础设施（无需改动）

| 链路 | 现状 | 结论 |
|------|------|------|
| 后端价格更新 API | `PUT /api/admin/products/{id}` 已支持 | ✅ 不需改动 |
| 小程序读取商品 | `GET /api/products` 每次查 DB | ✅ 实时最新价 |
| 官网读取商品 | `GET /api/products` 每次查 DB | ✅ 实时最新价 |
| 返现规则页面 | `GET /api/cashbacks/rules` 实时计算 | ✅ 实时最新价 |
| 下单锁价 | 创建订单时锁定 `totalAmount` | ✅ 已支付的订单不受影响 |

### 缺少的部分

| 缺失 | 说明 |
|------|------|
| 管理端前端编辑界面 | `ProductsPage.vue` 仅有展示和上下架，无编辑/创建功能 |
| 前端 API 函数 | `api.js` 缺少 `updateProduct()` |

## 修改计划

### 1. `api.js` — 新增接口函数

```js
export async function updateProduct(productId, payload) {
  const { data } = await client.put(`/admin/products/${productId}`, payload);
  return data.data;
}
```

### 2. `ProductsPage.vue` — 新增编辑弹窗

每个商品卡片增加「编辑」按钮 → 弹出玻璃拟态编辑弹窗 → 可编辑名称、价格、描述 → 保存后刷新列表。

弹窗样式：复用现有 `glass-panel`、`form-input`、`btn-primary` 等组件样式。
