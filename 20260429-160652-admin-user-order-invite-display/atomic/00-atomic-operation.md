# 原子化操作文档：管理端用户/订单/邀请展示增强

## 时间

## 目标
增强知禧管理后台的信息可读性：
- 用户管理显示用户头像。
- 订单管理显示下单时间时间戳和订单号。
- 邀请管理显示邀请人/被邀请人的头像、昵称、绑定时间戳、首单时间戳和邀请单号。

## 修改范围
- backend-api：Admin 用户 DTO 增加头像；新增 Admin 邀请 VO 并补充邀请双方资料聚合。
- zhixi-website/admin-frontend：用户、订单、邀请三个管理页展示字段与格式化。
- 不修改小程序业务逻辑。

## 已确认
- 订单后端模型已返回 `orderNo`、`createdAt`，前端缺展示。

- 邀请关系已有稳定 `id`、`boundAt`、`firstPaidAt`，可派生邀请单号 `INV-000001`。
