# 原子化待操作文档

## 目标

仅重构返现管理页面表格中“微信批次”和“微信明细”两列的展示样式，禁止长编号按字符垂直堆叠，超出宽度时单行省略。

## 修改范围

- admin-frontend/src/views/CashbacksPage.vue
- admin-frontend/src/styles.css

## 原子步骤

1. 为两列单元格改用专用 nowrap/ellipsis 渲染容器。
2. 新增 .text-ellipsis 原生 CSS 类：overflow hidden、text-overflow ellipsis、white-space nowrap、max-width 180px。
3. 增加仅在 DOM 实际溢出时返回 title 的辅助逻辑。
4. 不修改批准打款、确认收款、同步转账、复制确认参数等业务逻辑。
5. 运行 npm build 验证。

## 回滚方式

从本目录 code/zhixi-website 复制覆盖回 G:\zhiximini\zhixi-website，或使用远端备份仓库对应时间戳目录恢复。
