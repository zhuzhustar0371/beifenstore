## 四、修改内容

修改文件：

- `wechat-app/custom-tab-bar/index.wxss`

具体变更：

- `.liquid-tabbar` 高度从 `100rpx` 放大到 `128rpx`。
- `.liquid-tabbar` 横向内边距从 `26rpx` 放大到 `34rpx`。
- `.liquid-tabbar` 圆角从 `50rpx` 放大到 `64rpx`。
- `.tab-list` 横向内边距从 `26rpx` 放大到 `34rpx`。
- `.tab-item` 宽度从 `112rpx` 放大到 `148rpx`。
- `.tab-content` 宽度从 `112rpx` 放大到 `148rpx`。
- `.tab-content` 间距从 `16rpx` 调整为 `14rpx`，避免导航栏变高后上下过散。
- `.tab-bubble` 圆圈从 `50rpx x 50rpx` 放大到 `72rpx x 72rpx`。
- `.tab-bubble` 圆角从 `25rpx` 放大到 `36rpx`。
- `.tab-icon` 图标从 `25rpx x 25rpx` 放大到 `38rpx x 38rpx`。
- `.tab-label` 字号从 `20rpx` 放大到 `26rpx`。
- `.tab-label` 行高和最小高度从 `20rpx` 放大到 `28rpx`。
- `.tab-label` 最大宽度从 `112rpx` 放大到 `148rpx`。
- 激活态文字粗细从 `600` 加强到 `700`。
