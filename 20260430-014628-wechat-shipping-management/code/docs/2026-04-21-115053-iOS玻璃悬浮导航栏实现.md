## 1. 变更目标

- 将微信小程序底部导航栏调整为 iOS 风格的 Glassmorphism 悬浮胶囊导航。
- 采用高通透白色背景、毛玻璃模糊、细白边、多层柔和阴影和黄金比例式留白节奏。

## 2. 分析结论

- 当前底栏已有玻璃风格基础，但需要按用户给定规范收敛：
  - `backdrop-filter: blur(20px)`
  - `background: rgba(255, 255, 255, 0.75)`
  - `border: 1rpx solid rgba(255, 255, 255, 0.3)`
  - 悬浮底距约 `20px`
  - 高度固定，圆角等于高度一半
  - 多层柔和阴影
  - 图标圆圈做 1-2px 的视觉上移
  - 内边距与图标间距接近 `1 : 1.618`
- 本次只需要修改 `wechat-app/custom-tab-bar/index.wxss`。
