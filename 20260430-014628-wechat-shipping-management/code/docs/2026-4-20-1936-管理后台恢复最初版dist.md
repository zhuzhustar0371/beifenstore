## 注意事项

当前线上已经恢复到最初版 dist。

但是本地目录：

- `zhixi-website/admin-frontend`

仍是后续实验修改后的源码，并不等同于当前线上最初版 dist。如果后续再次在本地执行 `npm run build` 并部署，会再次覆盖原始布局。

因此后续如果要在最初版基础上继续增加：

- 中文枚举
- 批准打款按钮
- 返现管理操作

应先找回最初版对应的真实源码，或基于当前线上 `index-DpjZLsgc.js` 反向确认页面结构后重新实现，不应直接用当前本地 `admin-frontend` 覆盖线上。
