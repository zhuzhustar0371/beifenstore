## 参考仓库特征总结

- 参考仓库非常轻量，核心实现集中在：
  - `liquid-glass.js`
- 主要视觉语言包括：
  - 大圆角玻璃容器
  - 半透明高光层
  - 内阴影与外阴影叠加
  - `backdrop-filter`
  - 基于 SVG filter 的位移扭曲，制造液态折射感
- 这类实现路径适合网页环境，但不适合直接照搬到微信小程序原生 `tabBar`

## 当前项目限制判断

- 当前小程序底栏原本由 `wechat-app/app.json` 中的原生 `tabBar` 提供
- 原生 `tabBar` 限制较多：
  - 不能自由插入多层结构
  - 不能做浮动胶囊式玻璃容器
  - 不能实现参考仓库那种层叠高光与动态折射的复杂组合
- 因此如果要实现“液态玻璃风格”，必须改成小程序 `custom-tab-bar`

## 架构调整

- 在 `wechat-app/app.json` 中开启：
  - `"custom": true`
- 保留原有底部两个一级导航：
  - 首页
  - 我的
- 新增自定义底栏目录：
  - `wechat-app/custom-tab-bar/`

## 本次实现方案

- 不强行复制网页版的 SVG displacement 液态位移
  - 原因：小程序环境兼容性和维护成本都不合适
- 采用“视觉等价”方案来还原 liquid-glass 感受：
  - 浮动胶囊底栏外壳
  - 半透明白色渐变背景
  - 外部柔和阴影
  - 内部高光和内阴影
  - 左上与右下的环境光晕
  - 选中项使用抬起的玻璃气泡按钮
  - 文本和图标统一置中，保持苹果式悬浮导航的质感

## 关键实现细节

### 1. 自定义底栏容器

- 新底栏使用固定定位，悬浮在页面底部安全区之上
- 胶囊底栏本体采用：
  - 大圆角
  - 半透明渐变背景
  - 亮面高光层
  - 外部模糊阴影
  - 内部压痕式阴影

### 2. 液态玻璃视觉模拟

- 添加了两层环境光晕：
  - 左侧白色高光雾面
  - 右下偏青绿色散射光
- 选中项不是简单换色，而是通过一个抬起的玻璃泡泡强调层级
- 激活泡泡额外叠加了：
  - 更强高光
  - 更强投影
  - 更亮的玻璃背景
  - 轻微上浮位移

### 3. Tab 状态同步

- 因为切换到了 `custom-tab-bar`，需要手动同步当前选中项
- 已在两个 tab 页中补上：
  - `pages/index/index.js`
  - `pages/user/user.js`
- 页面 `onShow` 时会通过 `getTabBar().setData(...)` 更新当前索引

### 4. 页面底部留白

- 新底栏是浮动的，且高度明显高于原生底栏
- 如果不增加底部留白，页面内容会被遮挡
- 因此将两个 tab 页的底部 padding 增加为：
  - `220rpx`

- 没有直接复刻 SVG displacement 位移扭曲
- 主要复刻的是视觉语言，而不是网页侧的完整渲染管线
- 这是基于微信小程序运行环境限制做出的主动取舍

## 后续可继续优化的方向
