## 4. 本地代码修改
### 4.1 登录页：改为 getUserInfo 事件回调
文件：`G:\zhiximini\wechat-app\pages\login\login.wxml`
- 登录按钮改为：
  - `open-type="getUserInfo"`
  - `bindgetuserinfo="onGetUserInfo"`

文件：`G:\zhiximini\wechat-app\pages\login\login.js`
- 新增/调整逻辑：
  1. 将登录入口改为 `onGetUserInfo(e)`。
  2. 从 `e.detail.userInfo` 提取：`nickName/avatarUrl`。
  3. 保留原有勾选协议与后端可用性检查。
  4. 使用 `wx.login` 获取 code，提交至 `/api/auth/wechat-miniapp/login`。
  5. 登录成功时缓存 `userInfo.avatarUrl`（后端无值时回退到 `getUserInfo` 获取到的头像）。
  6. 移除 `wx.getUserProfile` 调用链路（避免 `scope unauthorized` 报错场景）。

### 4.2 我的页：使用 open-data 展示兜底
文件：`G:\zhiximini\wechat-app\pages\user\user.wxml`
- 头像展示改为三级兜底：
  1. `userInfo.avatarUrl`（优先）
  2. `<open-data type="userAvatarUrl">`（已登录且本地无头像）
  3. 默认头像图
- 昵称展示补充：
  - 增加 `<open-data type="userNickName">` 作为 `userInfo.nickname` 为空时的显示兜底。

### 4.3 用户信息刷新：保留头像字段
文件：`G:\zhiximini\wechat-app\pages\user\user.js`
- `refreshUserProfile` 的 `nextUserInfo` 增加：
