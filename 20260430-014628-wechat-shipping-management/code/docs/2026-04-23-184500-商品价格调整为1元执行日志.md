# 2026-04-23 商品价格调整为 1 元执行日志

## 1. 任务背景

- 时间：2026-04-23
- 需求：将商品价格调整为 `1.00` 元，返现比例规则不变。
- 关联截图：`D:/腾讯电脑管家截图文件/局部截取_20260423_185222.png`

## 2. 变更前分析

1. 线上接口 `https://api.mashishi.com/api/products` 显示商品 `id=1` 价格为 `0.10`。
2. 线上接口 `https://api.mashishi.com/api/cashbacks/rules` 显示返现比例为固定值：
   - 个人返现：第2单 10%，第3单 20%，第4单 70%
   - 邀请返现：第1批 100%，第2批及以后 20%
3. 当前系统规则由商品单价联动计算返现金额，因此只需改商品价格，不需改比例代码。

## 3. 执行动作

### 3.1 线上数据库执行调价

- 目标表：`products`
- 目标记录：`id = 1`
- 执行 SQL：

```sql
UPDATE products
SET price = 1.00
WHERE id = 1;
```

### 3.2 执行后核验

1. 数据库核验：
   - `SELECT id,name,price,status FROM products ORDER BY id`
   - 结果：`id=1` 价格已为 `1.00`
2. API 核验：
   - `GET https://api.mashishi.com/api/products`
   - 返回 `price: 1.00`
3. 规则联动核验：
   - `GET https://api.mashishi.com/api/cashbacks/rules`
   - 返回：
     - `unitPrice: 1.00`
     - 个人返现金额：`0.10 / 0.20 / 0.70`
     - 邀请返现金额：`1.00 / 0.20 / 0.20`
   - 比例字段保持不变（10% / 20% / 70% / 100% / 20%）。
